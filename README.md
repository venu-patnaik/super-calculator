# Super Calculator 🔢

A super-efficient calculator built in C++.

## 🚀 Features
- Basic arithmetic (add, subtract, multiply, divide)
- Clean terminal output
- Fast execution

## ⚙️ How to Run

1. Make sure you have `g++` installed.
2. Open terminal inside this folder.
3. Run:
   ```bash
   g++ main.cpp -o calculator
   ./calculator
